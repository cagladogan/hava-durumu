Bu proje, Python kullanılarak geliştirilmiş bir masaüstü hava durumu uygulamasıdır.
OpenWeatherMap API kullanarak seçilen şehrin güncel hava durumu bilgilerini getirir.
Türkiye’deki 81 ili SQLite veritabanında saklar ve alınan hava durumu bilgilerini
hava_durumu_bilgileri.txt dosyasına otomatik olarak kaydeder.
Özellikler

OpenWeatherMap API üzerinden canlı hava durumu verisi alma

81 şehir ve plaka bilgisi, SQLite veritabanında saklanır

Tkinter GUI ile modern bir masaüstü uygulaması

Sıcaklık, açıklama, ülke bilgisi ve simge gösterimi

Çekilen hava durumu bilgilerini .txt dosyasına loglama

“Yenile” butonu ile verileri güncelleme

Tarih ve saat gösterimi

Otomatik veritabanı oluşturma (sehirler.db
